
# Логика в действии

Готовый учебно-методический набор для изучения логических элементов и цифровых схем

«Логика в действии» — это комплексное решение для преподавателей информатики, технологии и специальных дисциплин. Набор включает аппаратную часть (реальные электронные компоненты) и полный методический комплект для проведения 10 готовых уроков. Мы превращаем абстрактную теорию из учебников в понятные, наглядные и безопасные практические занятия.

## О проекте

Проект родился из потребности дать ученикам и студентам возможность не просто механически решать логические уравнения, но и понимать, как логика работает в реальных цифровых устройствах. С помощью нашего набора можно не только строить электрические схемы и визуально решать уравнения, но и объяснить работу простейшей ячейки памяти — триггера.

Это готовый продукт «под ключ», который не требует от преподавателя дополнительной подготовки или глубоких знаний в электронике.

## Проблема

Для учеников и студентов:
- Решение логических уравнений носит формальный характер
- Отсутствие понимания практического применения изучаемых выражений
- Нет материалов для глубокого изучения темы

Для преподавателей:
- В кабинетах нет специализированного оборудования
- Нет времени на самостоятельную разработку уроков
- Закупка сложных промышленных стендов (от 50 000 руб.) не вписана в бюджет
- Бесплатные симуляторы не дают «живого» опыта работы со схемами

Последствия:
- Ученики не понимают тему
- Теряется интерес к инженерии и цифровой технике
- Уроки проходят «по учебнику» без практики
- Качество образования снижается

## Решение

Набор «Логика в действии» закрывает пустующую нишу между дешевыми игрушечными конструкторами (до 2000 руб.) и профессиональным оборудованием (от 50 000 руб.).

Наше ценностное предложение:
- Готовые уроки — преподаватель открывает коробку и проводит занятие, не тратя время на подготовку
- Наглядность — ученик видит результат сразу (светодиод загорается от его схемы), а не на экране
- Доступная цена — в 5–10 раз дешевле промышленных стендов
- Реальная схемотехника — настоящая пайка и логические микросхемы

## Состав набора

Аппаратная часть:
- Печатная плата с макетной областью

# Цифровой двойник доступного города: Навигатор для всех

Готовое решение для инклюзивной навигации и аудита городской среды

«Цифровой двойник доступного города» — это комплексная платформа, которая превращает абстрактное понятие «доступная среда» в работающий навигационный сервис и инструмент для улучшения города. Мы объединяем мобильное приложение для маломобильных групп, краудсорсинговую платформу для волонтёров и аналитическую систему для муниципалитетов в одном решении.

## О проекте

Проект родился из потребности дать людям с ограниченными возможностями, родителям с колясками и другим маломобильным группам реальный инструмент для безопасного перемещения по городу. Существующие навигаторы строят маршруты по скорости, игнорируя барьеры: лестницы, высокие бордюры, неработающие лифты. Мы создаём навигатор, который строит маршруты по доступности.

Это не просто приложение, а экосистема, которая объединяет пользователей, волонтёров и городские власти для совместного создания по-настоящему доступной городской среды.

---

## Проблема

Для людей с инвалидностью и родителей с колясками:
- Маршруты в 2ГИС и Яндекс Картах не учитывают наличие пандусов, ширину дверей и высоту бордюров
- Построенный маршрут часто оказывается непреодолимым
- Нет возможности узнать, работает ли лифт в переходе или не заставлен ли пандус машинами

Для муниципалитетов:
- Миллионные бюджеты тратятся на программу «Доступная среда» вслепую
- Нет актуальных данных о реальном состоянии инфраструктуры
- Непонятно, где действительно нужен пандус, а где он есть, но не работает

Для застройщиков и бизнеса:
- Нет простого инструмента для аудита объектов на доступность
- Сложно отслеживать соответствие нормам в динамике

Последствия:
- Город остаётся недоступным для тысяч людей
- Бюджетные средства расходуются неэффективно
- Социальная изоляция маломобильных групп усиливается

---

## Решение

Платформа «Цифровой двойник доступного города» закрывает пустующую нишу между общими навигаторами (2ГИС, Яндекс) и зарубежными картами доступности (Wheelmap), которые не адаптированы под российские реалии и не умеют строить маршруты.

Наше ценностное предложение:
- Для пользователей — первый навигатор, который действительно довезёт до цели без неприятных сюрпризов
- Для волонтёров — простой способ помочь городу: отметить точку на карте и внести вклад в социальную миссию
- Для муниципалитетов — готовая аналитика с тепловыми картами для программ «Доступная среда»
- Для застройщиков — объективная проверка объектов на соответствие нормам доступности

---

## Состав платформы

### 1. Мобильное приложение (для пользователей)

Бесплатное приложение с рекламной поддержкой, которое строит маршруты с учётом:
- Наличия пандусов и их угла наклона
- Ширины дверей и проёмов
- Высоты бордюров
- Работоспособности лифтов
- Тактильной плитки и озвучки светофоров
- Погодных условий (снег, гололёд, расчистка)
- Наличия шрифтов Брайля на зданиях

### 2. Режим разметки (краудсорсинг)

Любой пользователь или волонтёр может отметить на карте:
- «Здесь есть пандус»
- «Здесь бордюр 10 см»
- «Вход узкий»
- «Лифт не работает»

Геймификация: рейтинги, бейджи, «золотая коляска» за активность. Волонтёры — наш двигатель роста и актуализации данных.

### 3. Панель для муниципалитетов и застройщиков

- Тепловые карты проблемных зон
- Аналитические отчёты для планирования благоустройства
- Аудит объектов на соответствие нормам доступности
- Мониторинг изменений в динамике

- Расширение на 5–10 городов
- Запуск партнёрской программы с застройщиками
- Выход на годовую выручку более 6 млн руб.
